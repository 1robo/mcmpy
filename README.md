Three-day course during National Day  For ICC 


### 课前准备
- [**软硬件环境**]
    - Windows系统笔记本一台 
    - 安装python3.7以上版本 （参见python入门课程视频的第2课）
    - 安装matplotlib、numpy模块 
       - windows键+R 输入 “run" 进入命令行模式
       - python -m pip install -U matplotlib
       - python -m pip install -U numpy
